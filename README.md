# Lehenge Wala

### Sustainable Fashion Rental Platform (made for Innovate 3.0 Hackathon)

## Description

Lehenge Wallah is a peer-to-peer clothing rental platform where users can list their premium outfits and others can rent them. We handle pickup and delivery, making fashion cheaper, sustainable and circular. No more wasting money buying outfits you wear once!

### Features-

* List high quality clothing with images & details
* Browse outfits by style & owners
* Request rentals instantly
* Pickup & delivery support
* Earn money from unused wardrobe
* Promotes sustainable circular fashion
